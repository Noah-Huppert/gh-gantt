# Overview
GH Gantt frontend ReactJs single page web app.

# Dependencies
Install dependencies with [NPM](https://www.npmjs.com/):

```
npm install
```

# Build
Build with Webpack by running:

```
npm run build
```

To rebuild with webpack after each change run:

```
npm run watch
```
